# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1,
adapted for the `codex-app-web` community.

## Our Pledge

We pledge to make participation in this project welcoming, respectful, and
harassment-free for contributors, maintainers, users, and guests. We act in
ways that support an open, inclusive, and healthy community.

## Our Standards

Expected behavior includes:

- using welcoming and inclusive language;
- respecting different viewpoints and experiences;
- accepting constructive feedback gracefully;
- taking responsibility for mistakes and learning from them;
- focusing on what is best for the project and its users.

Unacceptable behavior includes:

- sexualized language or imagery, unwanted attention, or unwanted advances;
- insults, trolling, personal attacks, or derogatory comments;
- public or private harassment;
- publishing private information without explicit permission;
- conduct that would reasonably be considered inappropriate in a professional
  setting.

## Enforcement

Maintainers may remove, edit, or reject issues, pull requests, comments,
commits, discussions, or other contributions that do not align with this code.
Maintainers may temporarily or permanently restrict participation when needed to
protect the community.

Report unacceptable behavior privately to the repository owner through the
contact paths listed on the owner's GitHub profile. Do not open public issues
for conduct reports or include private personal information in repository
content.

Maintainers will review reports promptly and fairly, respect reporter privacy,
and choose proportionate actions such as clarification, warning, temporary
restriction, or permanent restriction.

## Scope

This code applies in repository spaces, GitHub Discussions, issues, pull
requests, and other project-managed channels. It also applies when someone is
officially representing the project in public.
